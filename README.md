CodeAlpha Stock Portfolio Tracker
Advanced Python Programming Internship Project
Advanced Stock Portfolio Tracker is a console-based Python application created as part of the CodeAlpha Python Programming Internship.

This project helps users manage a stock portfolio by buying stocks, selling stocks, checking investment value, calculating profit or loss, maintaining transaction history, and generating a portfolio report.

Project Objective
The objective of this project is to build a practical stock portfolio management system using Python.

The application demonstrates important Python programming concepts such as dictionaries, file handling, CSV storage, functions, loops, conditional statements, user input validation, arithmetic calculations, and report generation.

Features
View available stocks with fixed prices
Buy stocks and add them to portfolio
Sell stocks from existing holdings
View complete portfolio summary
Calculate total invested value
Calculate current market value
Calculate profit or loss
Search stock price by symbol
Maintain transaction history
Save portfolio data in CSV file
Generate a detailed text report
Easy-to-use menu-driven interface
Handles invalid input safely
